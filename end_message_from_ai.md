### Iteration 2

Extended the prototype for iteration 2 by adding TTML support without touching the QC engine/rules.

Changes made
Added new parser class: SubtitleQc.Core/Parsers/TtmlParser.cs
Implements ISubtitleParser
Supports XML namespace-agnostic TTML cue discovery via <p> nodes
Maps each TTML cue into existing internal Cue model
Resolves timing from:
begin + end
begin + dur
Supports common TTML time expressions:
clock times (hh:mm:ss, hh:mm:ss.fff)
offsets (ms, s, m, h)
Preserves line breaks from <br/> and nested text content
Uses stable fallback IDs (ttml-1, ttml-2, ...) when no id is present
